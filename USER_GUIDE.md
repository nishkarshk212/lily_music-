# 🎵 Lily Music Bot - User Guide

## ✅ Bot Status: WORKING

The bot is now optimized for **SoundCloud** - providing reliable, high-quality music playback!

## 🎧 How to Use

### Play Music
```
/play <song name>
```

**Examples:**
```
/play tere bina
/play blinding lights
/play shape of you
/play never gonna give you up
```

### How It Works
1. Bot searches SoundCloud automatically
2. Finds the best match for your song
3. Plays it instantly in voice chat

### Supported Features
✅ SoundCloud search & playback  
✅ Voice chat integration  
✅ Queue system  
✅ Skip, pause, resume controls  
✅ Now Playing display  
✅ High quality audio (192kbps MP3)  

## 💡 Tips for Best Results

1. **Be specific with song names:**
   - ✅ `/play tere bina lofi`
   - ✅ `/play shape of you ed sheeran`
   
2. **Use direct SoundCloud URLs (optional):**
   ```
   /play https://soundcloud.com/artist/song-name
   ```

3. **If a song doesn't work:**
   - Try a different spelling
   - Add artist name
   - Try adding "lofi" or "slowed" for remixes

## ❌ What Doesn't Work

YouTube playback is currently unavailable due to server network restrictions. The bot has been optimized to use SoundCloud exclusively for 100% reliability.

## 🎯 Commands

- `/play <song>` - Play a song
- `/skip` - Skip current song
- `/pause` - Pause playback
- `/resume` - Resume playback
- `/queue` - View song queue
- `/nowplaying` - Show current track
- `/stop` - Stop and clear queue
